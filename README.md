# Openflow

Scripts and setup material for [Openflow](https://docs.snowflake.com/en/user-guide/openflow/about) on Snowflake — a managed Apache NiFi runtime hosted in Snowpark Container Services (SPCS).

## Contents

| File | Description |
|------|-------------|
| `SPCS Setup.sql` | End-to-end SQL setup for deploying an Openflow deployment and runtime on Snowflake |
| `mssql_hol.py` | Hands-on lab helper — seeds a SQL Server database with demo data and generates a continuous order stream |
